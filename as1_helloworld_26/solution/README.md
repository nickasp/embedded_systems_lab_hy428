## Assignment 1: Hello World

This directory tree contains the skeleton of a very simple firmware program. Refer to [hy428](https://www.csd.uoc.gr/~hy428/as1.html) home page to find more information about the assignment. 

## How to use git

1. Fork the repo (Follow the instructions in [Assignments](https://www.csd.uoc.gr/~hy428/assignments.html)) 
2. Clone repo (git clone git@gitlab-csd.datacenter.uoc.gr:`<csdxxxx>`/as1_helloworld.git)
3. **Change** the permissions to **PRIVATE** (you can do it when forking the repository).

## Change the permissions

1. Go to Project Settings -> General
2. Visibility
3. Change Project Visibility to Private

## Add the TA in your project

1. Go to Project Information -> Members
2. Add each TA as a Developer 

    TA_1:  csdp1486

## Submission 

- You should submit the **source files** of each assignment. **Do not** submit the toolchain and object or executable file
- Commit all files (not executables or object files!) 

```
git add file1.c file2.c file3.c  
git commit -m "Commit message"  
git push  
```

- Make sure that your latest commit is uploaded on gitlab!
