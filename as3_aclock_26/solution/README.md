## How to use git

1. Fork the repo (Follow the instructions in [Assignments](https://www.csd.uoc.gr/~hy428/assignments.html))
2. Clone repo (git clone git@gitlab-csd.datacenter.uoc.gr:`<csdxxxx>`/as3_aclock_26.git)
3. **Change** the permissions to **PRIVATE** (you can do it when forking the repository).

## Change the permissions

1. Go to Project Settings -> General
2. Visibility
3. Change Project Visibility to Private
